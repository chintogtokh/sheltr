# Sheltr app

Sheltr - connecting safe homes with new students.

This app uses the MERN stack.

## Installation

Running the app: `nf start`


## API reference

* `/auth/login`
* `/auth/register`

* `/suburbs` - returns all suburbs
* `/suburb/:SUBURB_SHIM` - returns certain suburb
